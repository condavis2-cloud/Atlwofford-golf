📖 HOW TO USE THIS SITE
Written for anyone — no tech skills needed.
🔑 How to get into Admin mode
Click the ⚙️ gear icon in the top right corner of the nav bar. Type the password when it asks. That's it — you're in.
✏️ How to change any text on the page
In Admin mode, click directly on any text — headings, descriptions, numbers, anything. It turns into a text box. Type your change. Click somewhere else when done. Then hit Save.
💾 How to save so everyone sees your changes
Click the big gold "Save Changes" button at the top of this panel. Done. Everyone who opens the site sees your update immediately — no downloading, no uploading, nothing extra.
🏌️ How to add a sponsor
1. Scroll to the Sponsors section on the page
2. Find the right level (Gold, Silver, Bronze, etc.)
3. Click the dashed "+ Add..." button
4. Type the name → click Save
5. Hit the gold Save button up top
⛳ How to fill in a team
1. Scroll to the Teams section
2. Find a red "Open Slot" card
3. Click "+ Fill Slot"
4. Type the captain's name, then player names (one per line)
5. Click Save → then hit the gold Save button up top
🎁 How to edit raffle prizes
Click the Raffle tab at the top of this panel → edit any prize name → click "Update Prizes on Page" → then hit the gold Save button.
📋 How to track open tasks
Click the Actions tab. You'll see all the open tasks. Click any field to edit it. Change the dropdown between ⏳ Open, ✅ Done, and 🚫 Blocked. Hit Save when done.
⚙️ How to change the name, date, or other site details
Click the Settings tab → change what you want → click "Apply All Changes" → hit the gold Save button.
🚪 How to leave Admin mode
Click "← Exit Admin View" or the ✕ at the top. The site goes back to normal. Nothing breaks. Come back anytime.
🆘 Something looks wrong — what do I do?
Click "⬇ Download Backup File" in the Dashboard tab, then drag that file into the Claude chat window. Claude will fix it and send it right back.
